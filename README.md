---

## ⚙️ Funcionalidades

- 🕒 Horarios por dirección  
- ⏱️ Cálculo de próxima salida  
- 🚌 Seguimiento visual del último bus salido  
- 📡 Reportes comunitarios en tiempo real  
- 🔔 Notificaciones desde Firestore  
- 📍 GPS para detección de parada más cercana  
- 🛠️ Panel de administración  

---

## 🚀 Deploy

Este proyecto es compatible con cualquier hosting estático:

- Vercel  
- Netlify  
- Firebase Hosting  

Solo necesitas subir los archivos del proyecto.
